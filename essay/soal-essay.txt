4. Apa itu state dan props? Apa perbedaannya?
Perbedaan keduanya:
-> State adalah data yang lahir pada sebuah komponen, dan dapat diubah (mutable). 
-> Props berguna untuk mengoper data dari parent component ke child component, tidak dapat diubah (immutable), bersifat satu arah, dan dapat menjadikan komponen lebih dinamis. Props dapat digunakan untuk mengoper data: string, number, boolean, array, object dan function.

6. Apa itu state management dan kapan harus menggunakannya?
-> State management merupakan teknik yang digunakan dalam menyimpan, mengelola dan mengakses data yang sifatnya sering berubah, seperti profil pengguna dan postingan terbaru.

7. Apa itu ORM/ODM?
-> ORM (object-relational mapping), teknik menyelaraskan kode pemrograman berbasis objek (OOP/objct-oriented programming) dengan struktur database seperti relational database (sumber: revou.com).
-> ODM (object-document mapping), merupakan tools yang memungkinkan kita untuk berinteraksi dengan MongoDB database menggunakan javascript object (sumber: dctacademy.com).

8. Apa perbedaan SQL dan NoSQL?
Perbedaan SQL dan NoSQL:
-> Sql memiliki struktur data yang relasional, maksudnya data disimpan dalam bentuk tabel dengan kolom dan baris yang mana struktur tabel sudah terlebih dahulu didefenisikan sebelum dapat diinput.
-> Nosql mekiliki sturktur data yang lebih fleksibel dan tidak memiliki tabel, yang mana datanya disimpan dalam dokumen, grafik atau key-value store.
